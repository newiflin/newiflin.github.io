Chrome 快捷键

发表于 2019-11-20 更新于 2020-01-19 分类于 快捷键

前言

Chrome ，即谷歌浏览器，英文直译为化学元素”铬“，是由 Google 公司开发的一款免费的网页浏览器。于2008年9月首次发布后，Chrome 用户数量迅速增长，仅用了4年时间，市场占有率便超越了 FireFox 和 IE，截止19年， Chrome 浏览器已占有将近 7 成的市场份额。对于当下最广为流行的浏览器，学习掌握其特性和使用自然是完全有必要的。

Chrome 快捷键

标签页和窗口

快捷键 操作
Ctrl+N 打开新窗口
Ctrl+T 打开新的标签页
Ctrl+Shift+N 在隐身模式下打开新窗口
Ctrl+Shift+T 恢复最近关闭的标签页（最多10个）
Ctrl+O 选择打开计算机上的文件
Ctrl+点击链接 新标签页打开链接，仍停留在原页面
Ctrl+Shift+点击链接 新标签页打开链接，同时切换到新页面
Shift+点击链接 在新窗口打开链接
Ctrl+Tab或Ctrl+PgDown 切换到下一个标签页
Ctrl+Shift+Tab 或 Ctrl+PgUp 切换到上一个标签页
Alt+←/→ 历史记录上一页/下一页
Ctrl+W 或 Ctrl+F4 关闭当前的标签页
Ctrl+1到Ctrl+8 切换到指定位置编号的标签页
Ctrl+9 跳转到最后一个标签页
Alt+Home 在当前标签页打开主页
Ctrl+Shift+W 关闭所有标签页
Ctrl+Shift+Q 退出浏览器
Alt+F4 关闭当前窗口
Alt+Space+C 关闭当前窗口

地址栏

快捷键 操作
Ctrl+Enter 自动为地址输入内容添加 “www” 和 ”.com“，然后打开网址
Enter 使用搜索引擎进行搜索
Alt+Enter 在新标签页打开网址
Alt+D 聚焦或选中地址栏中的内容
Ctrl+L 聚焦或选中地址栏中的内容
F6 聚焦或选中地址栏中的内容
Ctrl+←/→ 光标移到上 / 下一个关键字词
Ctrl+Space 删除地址栏中光标前的字词
输入搜索引擎关键字或网址+Tab 使用相关联的搜索引擎进行搜索

网页

快捷键 操作
Ctrl+P 打印当前页
Ctrl+S 保存当前页
F5 重新加载当前页
Ctrl+R 重新加载当前页
Esc 停止加载当前页
Ctrl+F 打开查找栏
点击鼠标滚轮 激活自动滚动。根据鼠标的移动方向控制滚动方向，位移量控制滚动速度
Ctrl+F5 或 Shift+F5 重新加载当前页，清除缓存内容
Alt+ 点击链接 下载链接
Ctrl+G 或 F3 查找与页面搜索相匹配的下一匹配项
Ctrl+Shift+G 或 Shift+F3 查找与页面搜索相匹配的上一匹配项
Ctrl+U 查看源代码
Ctrl+D 将当前网页加入书签
Ctrl+Shift+D 将所有打开的网页以书签形式保存在新文件夹
Ctrl++ 放大网页上的所有内容
Ctrl+- 缩小网页上的所有内容
Ctrl+0 将网页上的所有内容都恢复到正常大小
Home 转至网页顶部
End 转至网页底部
Space 向下滚动页面
↓/↑ 向下 / 上 滚动页面
Shift+↑/↓ 向右/左横向滚动页面
F11 启动/取消全屏模式

功能

快捷键 操作
Ctrl+Shift+J 或 F12 打开开发人员工具
Ctrl+Shift+Delete 打开”清除浏览数据”对话框
F1 在新标签页中打开帮助中心
Tab 下一个属性
Enter 执行命令
Alt+E 打开工具菜单
Ctrl+B 打开和关闭书签栏
Ctrl+Shift+`B`` 打开书签管理器页
Ctrl+H 在新标签页打开”历史记录”页
Ctrl+J 在新标签页打开”下载内容”页
Shift+Esc 查看任务管理器
Shift+Alt+T 将焦点设置在工具栏上

文本

快捷键 操作
Ctrl+C 将选中内容复制到剪贴板
Ctrl+Insert 将选中内容复制到剪贴板
Ctrl+V 从剪贴板粘贴内容
Shift+Insert 从剪贴板粘贴内容
Ctrl+Shift+V 从剪贴板粘贴内容（不带格式）
Ctrl+X 剪贴选中内容到剪贴板
Shift+Delete 剪贴选中内容到剪贴板

Vimium 快捷键

Vimium 简介

Vimium 是 Chrome 的一款快捷键扩展插件，继承了 Vim 中的常用键位，相比Chrome 拥有更为简单的快捷键，让用户在使用 Chrome 过程中几乎可以完全脱离鼠标，全通过键盘完成浏览网页、切换标签或是其他任何操作。

浏览网页快捷键

快捷键 操作
j 向下细微滚动页面
k 向上细微滚动页面
gg 到页面顶部
G 到页面底部
d 向下滚动半页
u 向上滚动半页
h 页面向左细微滚动
l 页面向右细微滚动
r 刷新页面
yy 复制当前 URL 到剪贴板
yf 拷贝页面某一个 URL 到剪贴板
p 在当前页打开剪贴板 URL
P 在新标签页打开剪贴板 URL
i 进入插入模式
v 进入虚拟模式
gi 聚焦页面第一个文本输入框
f 在当前页面打开一个链接（生成快捷键）
F 在新标签页打开一个链接（生成快捷键）
gf 选中当前页面页面下一个框层
gF 选中当前页面的主框层

搜索条

快捷键 操作
o 当前页面打开 URL, 书签或者历史记录
O 新标签页打开 URL，书签或者历史记录
b 在当前页面打开一个书签
B 在新标签页打开一个书签
T 在开启的标签中查找

查找

快捷键 操作
/ 进入查找模式
n 下一个匹配内容（循环）
N 上一个匹配内容

导航历史

快捷键 操作
H 后退历史
L 前进历史
gu 跳转到父页面

操作标签页

快捷键 操作
t 创建新标签页
gT 切换到左边的标签页
gt 切换到右边的标签页
^ 切换到上次访问的标签页
g0 切换到第一个标签页
g$ 切换到最后一标签页
yt 重复当前标签页
x 关闭当前标签页
X 恢复当前标签页
gs 查看网页源码

帮助

快捷键 操作
? 显示帮助
