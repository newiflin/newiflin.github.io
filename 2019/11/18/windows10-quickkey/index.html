windows 10 常用快捷键

发表于 2019-11-18 更新于 2019-11-19 分类于 快捷键

前言

Windows 10 是美国微软公司研发的跨平台及设备应用的操作系统，是微软的最后一个独立 Windows 版本，也是目前最普遍、市场份额最大的操作系统。善于使用键盘快捷键，省去额外鼠标滑动和点击操作，能够有效减少操作时间。本文将尽可能搜集、整理及更新最新版 Win10 的快捷键。

微软官网 Win10 介绍

Windows 10 快捷键

Win 相关快捷键

快捷键 操作
Win+A 打开操作中心
Win+B 将焦点转移到任务栏通知区域
Win+D 显示桌面
Win+E 打开资源管理器
Win+G 打开游戏录制工具
Win+H 通过语音激活 Cortana
Win+I 打开 Windows 设置应用
Win+K 激活无线显示器连接或音频设备连接
Win+L 锁定屏幕
Win+M 最小化所有窗口
Win+O 打开或关闭屏幕旋转锁定
Win+P 打开投影，可用于复制、扩展显示屏幕
Win+Q 弹出索引搜索
Win+R 打开运行
Win+S 打开索引搜索
Win+T 将焦点切换到任务栏图标或在图标之间切换
Win+U 打开 Windows 设置 — 轻松使用
Win+V 查看剪切板历史记录（体验复制粘贴的快乐）
Win+W 打开 Windows Ink（含白板、全屏截图功能，可编辑）注：新版 win10 在 Windows Ink 中去掉了便笺
Win+X 激活开始按钮右键菜单
Win+Ctrl+D 新建虚拟桌面
Win+Ctrl+F 打开查找计算机窗口
Win+Ctrl+F4 关闭正在使用的虚拟桌面
Win+Ctrl+←/→ 在已打开的虚拟桌面之间进行切换 （BOSS键）
Win+Shift+M 取消最小化所有窗口
Win+Shift+S 自定义（矩形、任意形状、窗口、全屏）截图，自动保存到剪贴板！！！缺点：不可编辑
Win+Shift+↑ 在垂直方向最大化窗口
Win+Shift+←/→ 将应用窗口移动到左边或右边的显示器中
Win+Alt+G 录制游戏最近 30 秒
Win+Alt+R 开始录制游戏
Win+Alt+Prtsc 对游戏进行截屏
Win+Alt+数字键 与右击任务栏对应位置应用效果相同
Win+Space 切换输入法
Win+Pause 打开系统属性窗口
Win+, 窥探桌面
Win+. 打开表情符号
Win+Tab 激活任务视图（显示所有已打开的应用和桌面）
Win+= 启动放大镜或放大屏幕内容
Win+- 放大镜启动后，缩小屏幕内容
Win+Home 除了当前窗口，将其他窗口最小化/恢复
Win+Esc 放大镜开启状态时，退出放大镜
Win+数字键 打开任务栏上固定的应用或程序
Win+Prtsc 截全屏并保持到用户图片 \ 屏幕截屏文件夹下
Win+↑/↓ 使应用窗口在最大化，正常状态以及最小化之间进行切换
Win+←+→ 使应用窗口在占据左/右半边屏幕以及正常状态之间切换
Win+←+→ > Win+↑/↓ 使应用窗口占据屏幕四个角落 1/4 的屏幕区域（二分屏、三分屏、四分屏）
Win+F1 启动 Edge 搜索 “如何在 windows 10 中获取帮助”
Win+Ctrl+Shift+B 重启显卡驱动 【解决部分假死机问题】

Ctrl 相关快捷键

快捷键 操作
Ctrl+Shift 输入法切换
Ctrl+Shift+N 新建文件夹【文件资源管理器】【桌面】
Ctrl+Shift+Esc 启动任务管理器
Ctrl+Space 中英文切换
Ctrl+Home 光标快速移到文件头
Ctrl+End 光标快速移到文件尾
Ctrl+Esc 显示开始菜单
Ctrl+鼠标拖动文件 复制文件
鼠标拖动文件+Ctrl+Shift 创建文件的快捷方式
Ctrl+A 选中全文
Ctrl+C 将选中的文字复制到剪贴板中
Ctrl+F 打开“查找与替换”对话框，并定位在“查找”
Ctrl+H 打开“查找与替换”对话框，并定位在“替换”
Ctrl+N 创建同级界面
Ctrl+O 激活“打开”对话框【编辑器】【浏览器】
Ctrl+P 激活“打印”对话框 【编辑器】【浏览器】
Ctrl+V 将剪贴板中的文本或图片粘贴到光标处
Ctrl+W 关闭当前文档
Ctrl+X 将选中的文字剪切到剪贴板中
Ctrl+Y 恢复上一步操作
Ctrl+Z 撤销上一步操作
Ctrl+End 快速将光标移到文末
Ctrl+Home 快速将光标移到文首

Alt 相关快捷键

快捷键 操作
Alt+A 截图（前提：启动微信 PC 版）
Alt+Space+C 关闭窗口
Alt+Space+M 移动窗口
Alt+Space+N 最小化当前窗口
Alt+Space+R 恢复窗口
Alt+Space+S 改变窗口大小
Alt+Space+X 最大化当前窗口
Alt+Tab 切换活动的应用
Alt+Enter 查看文件属性
Alt+Double Click 查看文件属性
Alt+255 QQ 号中输入无名人
Alt+F4 关闭 Windows【桌面】或关闭窗口
Alt+↑ 打开上一级界面 【文件资源管理器】
Alt+← 打开上一次界面 【文件资源管理器】
Alt+PrtScn 窗口截图

Shift 相关快捷键

快捷键 操作
Shift 切换输入法英文/中文
Shift+字母 输入法字母大写
Shift+Ctrl+Esc 打开任务管理器
Shift+Delete 永久删除
Shift+Space 半 \ 全角切换
Shift+右击 右击菜单打开方式
Shift+F10 选中文件的右菜单
Shift+多级文件 全部关闭
Shift+任务栏图标 新建一个该应用窗口

其他快捷键

快捷键 操作
F2 对文件重命名（等效于鼠标慢文件名两次）
Delete 删除文件 → 回收站
PrtScn 全屏截图

运行窗口命令

命令 操作
cmd 启动命令提示符
osk 打开虚拟键盘
notepad 启用记事本
psr.exe 打开步骤记录器
calc 打开计算器
shell:startup 打开程序启动目录
