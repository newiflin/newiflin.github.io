VS Code windows10 快捷键

发表于 2019-10-23 更新于 2019-10-26 分类于 快捷键

前言

Visual Studio Code（简称 VS Code）是一款由微软推出的开源代码编辑器，支持多种编程语言，集成终端，可以在编译器中脚本、编译软件、调试脚本、设置断点、做版本管理。此外VS Code 还支持丰富的扩展程序。VS Code 凭借众多优点已一跃成为”最受欢迎的开发环境“，没有之一！

掌握 VS Code 的快捷键，对于开发效率的提升将有很大帮助。

VS Code 快捷键

通用部分

快捷键 命令
Ctrl+Shift+P, F1 显示命令
Ctrl+P 快速打开文件
Ctrl+Shift+N 新建窗口
Ctrl+Shift+w 关闭窗口

基本的编辑

快捷键 命令
Ctrl+X 剪切选中的内容或剪切行（空选）
Ctrl+C 复制选择的内容或复制行（空选）
Alt+↑/↓ 向上/向下移动一行代码
Shift+Alt+↑/↓ 复制当前行，向上/向下粘贴
Ctrl+Shift+K 删除当前行
Ctrl+Shift+\ 跳到匹配的括号
Ctrl+Shift+]+[ 增加/减小一个缩进
Home 跳到当前行开始位置
End 跳到当前行结束位置
Ctrl+Home 跳到文件的开始位置
Ctrl+End 跳到文件的结束位置
Ctrl+↑/↓ 向上 / 下滚动一行
Alt+PgUp/PgDown 向上 / 下滚动一页
Ctrl+Shift+[ 折叠区域
Ctrl+Shift+] 展开区域
Ctrl+K Ctrl+[ 折叠所有子区域
Ctrl+K Ctrl+] 展开所有子区域
Ctrl+K Ctrl+J 展开所有区域
Ctrl+K Ctrl+C 添加行注释
Ctrl+K Ctrl+U 移除行注释
Ctrl+/ 添加或移除行注释
Shift+Alt+A 添加或移除多行注释
Alt+Z 适应或取消适应屏幕

导航栏

快捷键 命令
Ctrl+T 显示所有的符号
Ctrl+G 跳到指定行
Ctrl+P 跳到指定文件
Ctrl+Shift+O 跳转到指定符号
Ctrl+Shift+M 显示问题面板
F8 跳到下一个错误或警告
Shift+F8 跳到上一个错误或警告
Ctrl+Shift+Table 导航编辑器组历史记录
Alt+←/→ 退回/向前一个操作的位置
Ctrl+M 启动或取消标签移动焦点

查找或替换

快捷键 命令
Ctrl+F 文件内查找
Ctrl+H 文件内替换
F3/Shift+F3 查找下一个/上一个
Alt+Enter 选中所有查找匹配到的内容
Ctrl+D 选中下一个匹配到的内容
Ctrl+K Ctrl+D 移除上次选中，跳到下一个匹配
Alt+C/R/W 开启或取消大小写敏感/正则/全局搜索

多光标和选择

快捷键 命令
Alt+Click 插入光标
Ctrl+Alt+↑/↓ 向上/向下插入光标
Ctrl+U 撤销上一次光标操作
Shift+Alt+I 在选择的每一行末尾插入光标
Ctrl+L 选中当前行
Ctrl+Shift+L 选中所有与当前选择相同的项
Ctrl+F2 选择所有与当前光标所在单词相同的项
Shift+Alt+→ 扩展选择
Shift+Alt+← 缩减选择
Shift+Alt+(drag mouse) 列选择
Ctrl+Shift+Alt+(arrow key) 列选择
Ctrl+Shift+Alt+PgUp/PgDown 列选择向上/下一页

富语言编译

快捷键 命令
Ctrl+Space 触发建议
Ctrl+Shift+Space 触发参数提示
F12 跳转到定义处
Alt+F12 查看定义
Shift+F12 查看引用
F2 重命名
Ctrl+Shift+. 树形显示文件内容
Ctrl+K Ctrl+X 截取掉尾部空格

编辑器管理

快捷键 命令
Ctrl+F4, Ctrl+w 关闭编辑器
Ctrl+K F 关闭文件夹
Ctrl+\ 分割编辑器
Ctrl+1/2/3 聚焦第1/2/3编辑器组
Ctrl+K Ctrl+←/→ 聚焦前一个/后一个编辑器组
Ctrl+Shift+PgUp/PgDown 向左/右移动编辑器

文件管理

快捷键 命令
Ctrl+N 新建文件
Ctrl+O 打开文件
Ctrl+S 保存文件
Ctrl+Shift+S 文件另存为
Ctrl+K S 保存所有文件
Ctrl+F4 关闭
Ctrl+K Ctrl+W 关闭所有文件
Ctrl+Shift+T 重新打开关闭的编辑器
Ctrl+Tab 切换到下一个打开的文件
Ctrl+Shift+Tab 切换到上一个打开的文件
Ctrl+K P 复制文件的目录
Ctrl+K R 在资源管理器显示所有活跃的文件
Ctrl+K O 在新的窗口显示活跃文件

显示

快捷键 命令
F11 启动或取消全屏显示
Ctrl+=/- 放大/缩小
Ctrl+B 显示/隐藏侧边栏
Ctrl+Shift+E 显示资源管理器/开启或取消文件聚焦
Ctrl+Shift+F 显示搜索
Ctrl+Shift+G 显示Git
Ctrl+Shift+D 显示Debug
Ctrl+Shift+X 显示扩展
Ctrl+Shift+H 替换文件中的匹配上的内容
Ctrl+Shift+J 开启或取消搜索细节
Ctrl+Shift+C 打开命令提示符
Ctrl+Shift+U 显示输出面板
Ctrl+Shift+V 开启或取消markdown预览
Ctrl+K+V 右侧边开启markdown预览

Debug

快捷键 命令
F9 设置或取消断点
F5 开始/继续
Shift+F5 停止
F11/Shift+F11 进入/跳出
F10 单步执行
Ctrl+K Ctrl+1 显示悬停

集成终端

快捷键 命令
Ctrl+` 显示集成终端
Ctrl+Shift+` 创建一个新的终端
Ctrl+Shift+C 复制选中的内容
Ctrl+Shift+V 粘贴到激活状态的终端
Ctrl+↑/↓ 向上/下滚动
Shift+PgUp/PgDown 向上/下滚动一页
Ctrl+Home/End 滚动到顶部/底部

获取官方快捷键文件

Ctrl+K Ctrl+R 启动浏览器自动打开一篇包含 VS Code 支持快捷键的 pdf。
