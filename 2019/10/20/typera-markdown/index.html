Typora 的基本语法基础

发表于 2019-10-20 分类于 markdown

前言

Typora 是当下一款流行的 mardown 编辑器和阅读器，简单、高效。

本篇是对 Typora 帮助使用文档中语法部分的整理，方便日后查阅和熟悉。

块级元素

段落和换行

  • 单击 return，创建一个新的段落；
  • 点击 Shift+Return 创建单行换行 ，兼容大多数其他的markdown转换器，末尾留两个空格；
  • 或者插入 <br/>

标题

  • 标题在行首前使用1~6个 # 字符，相当于header的 h1 到 h6。

引用

Markdown 使用 email 风格的 > 字符作为块级应用。如下呈现：

这是两段块级引用。

这是第一段落。

这是第二段落。

这是一段另一个块级引用。

列表

  • 输入 * 列表项 将创建未排序的列表，也可以使用 + 或 - 代替 *；
  • 输入1. 列表项 将创建有序列表。

无序列表实例：

  • 红色
  • 绿色
  • 蓝色

有序列表实例：

  1. Red
  2. Green
  3. Blue

任务列表

任务列表用 - [ ] 或 - [x] （未完成或已完成）标记。

  • 任务列表项
    • 未完成
    • 已完成

代码块

输入```，然后点击回车，选择一门语言，然后可以看到语法高亮。

数学块

  • 可以使用 MathJax 提取 LaTex 数学表达式；
  • 输入$$，添加数学表达式，输入回车。

$$
\mathbf{V}_1 \times \mathbf{V}_2 = \begin{vmatrix}
\mathbf{i} & \mathbf{j} & \mathbf{k} \
\frac{\partial X}{\partial u} & \frac{\partial Y}{\partial u} & 0 \
\frac{\partial X}{\partial v} & \frac{\partial Y}{\partial v} & 0 \
\end{vmatrix}
$$

表格

输入 | First Header | Second Header|，再输入回车。将创建一行两列的表格。

创建出的列表，可以使用工具条重新调整表格大小、对其或者删除，也可以使用文本菜单复制、删除、添加行列。Typora 自动生成，没有必要完全了解markdown表格源码。

表格中可以包括连接、粗体、斜体或者删除线。

第一个标题 第二个标题
内容1 内容2

注脚

使用[^footnote]创建注脚。

这是一个注脚[^hello]

水平线

在空行输入 *** 或 --- ，再回车就可以画出一条水平线。

YAML前页

Typora支持 YAML 前页。在文章顶部输入 --- ，回车后引入 metadata 块。

目录表

输入 [toc]，按下 enter 键，根据文档标题自动生成、更新目录表。

[TOC]

内联元素

链接

[显示的名称](实际的链接)

这是百度的链接。

  • 内部链接

​ 可以设置超文本引用到标题，将创建一个允许点击后跳转到对应部分的书签。

​ windows: ctrl+click 链接，将跳转到块级元素标题。

  • 参考链接

​ 使用第二个方括号[]，内部放了你选择定位的链接

这是一个参考链接

​ 在文档的任何地方，都可以像这样定义：

​ 两个方括号的内容名相同，可以省略第二方括号的名字

​ Google

​ 然后定义链接：

注意：在 Typora 中，点击链接只是看到源码，ctrl + 鼠标左键，才能启动浏览器打开链接。

URLs

Typora可允许直接将 URLs 作为链接，使用 <url>。

例如：

<www.google.com>

图片

语法类似于链接，只不过在前面多了一个！，也可直接将图片拖拽进 Typora。右键图片可进行缩放比设置。

强调

使用 * 和 _ 作为强调指示符（斜体）。文本包含在单一的*或_将被HTML标签包含。

加强

两个 * 或 _ 包围文本，等价于HTML标签包围。

双星号

双下划线

代码

内联代码，使用反单引号包含单行代码。

use the printf() function.

删除线

双波浪号 ~ 包含文本。

错误的文本

下划线

使用原生的HTML产生，<u>添加下划线的文本</u>。

下划线

表情

  • 输入表情语法 :smile:。
  • 智能提示表情，ESC键退出。

:smile::black_flag:

内联数学

通过首选项->Markdown，开启功能。使用 $ 包含 Tex 命令。

$\lim_{x \to \infty} \exp(-x) = 0$

下标

同样需要开启扩展语法。然后使用 ~ 包含下标文本。

H2O

上标

开启扩展语法，使用 ^。

X^2^

高亮

开启扩展语法，使用==。

==highlight==

HTML

可以使用HTML，不过原生Markdown不支持。

this text is red

强调内容

iframe 标签。

视频

video 标签引入视频。

其他HTML支持

可参考相关文档，获取更多特性。
